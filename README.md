# <img src="./manual/app-icon.png" width="28"> [fBanner](https://finfra.kr/product/fBanner/en/index.html)

> **The moment you copy, your banner image is ready.**

macOS Menu Bar App for instant banner image generation. Copy a URL or text, and fBanner automatically creates a banner image for you.

## Features

- **Instant Access** - `Cmd+Shift+K` shortcut for immediate access
- **Clipboard Automation** - Auto-detects copied URL/text
- **Versatile Formats** - PNG download, grid layout adjustment
- **Menu Bar Integration** - Lives quietly in your system menu bar
- **Generation History** - Reuse previously generated images
- **Multi-language Support** - English, Korean, Japanese, Chinese

## Requirements

- macOS 14.0 or later
