# 용어 사전 (Glossary)

이 문서는 fBanner 프로젝트 내의 주요 용어와 다국어(8개 언어) 번역을 정리한 문서입니다. 개발 규칙, 매뉴얼 작성, UI 라벨링 시 이 용어를 기준으로 합니다.
